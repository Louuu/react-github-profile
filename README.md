# GitHub Profile Page

This is a sample React project I made which pulls data from the GitHub API.

## Installation

This application uses [HashiCorp Waypoint](https://www.waypointproject.io/) to build and deploy to a local Docker install. You will need to install Waypoint onto your Docker host if you have not done so already.

```bash
waypoint init
waypoint up
```


## Development

```bash
npm install
npm start
```
